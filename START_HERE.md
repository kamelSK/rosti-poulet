## ⚡ Étapes de mise en ligne (5 min)

### Étape 1 : Remplacer 3 choses
Dans TOUS les fichiers, remplacer :
- `06 XX XX XX XX` → Votre téléphone
- `contact@rostipoulet.fr` → Votre email
- `www.rostipoulet.fr` → Votre domaine

✅ **Conseil** : Utiliser Ctrl+H pour chercher/remplacer rapidement

### Étape 2 : Google My Business
1. Allez sur https://maps.google.com
2. Trouvez votre établissement
3. Cliquez "Partager" → "Intégrer"
4. Copiez le code iframe
5. Remplacez l'iframe dans **index.html**

### Étape 3 : Uploader sur hébergeur
1. FTP ou cPanel
2. public_html dossier
3. Uploader TOUS les fichiers (HTML, images, config)

✅ **C'est fait !** Votre site est en ligne ! 🎉

## 🆘 Les 3 problèmes courants & solutions

### Problème 1 : Images ne s'affichent pas
```
✅ Solution : Vérifier que les noms exacts
   (unnamed.webp, unnamed (1).webp, etc.)
   sont présents et uploadés
```

### Problème 2 : Site très lent
```
✅ Solution : 
   1. Uploader le .htaccess
   2. Compresser les images
   3. Attendre le cache
```

### Problème 3 : Pas de trafic SEO
```
✅ Solution :
   1. Attendre 4-8 semaines (normal)
   2. Ajouter du contenu frais
   3. Optimiser pour mots-clés
```
